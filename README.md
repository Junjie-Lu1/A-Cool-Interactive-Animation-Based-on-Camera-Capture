# 一个基于摄像头的交互酷炫动画
要实现这个效果，你只需要按照下列步骤进行：
 - 下载 [Cool.html](https://github.com/Junjie-Lu1/A-Cool-Interactive-Animation-Based-on-Camera-Capture/blob/main/Cool.html) 放到你自己的文件夹里；
 - 打开文件夹，右键，选择 “打开方式”；
 - 选择 Google Chrome 打开。

是不是很简单！快去试试！

**注意**：需要给 Google Chrome 授权摄像头权限。

---

To achieve this effect, you just need to follow these steps:
 - Download [Cool.html](https://github.com/Junjie-Lu1/A-Cool-Interactive-Animation-Based-on-Camera-Capture/blob/main/Cool.html) and place it in your own folder;
 - Open the folder, right-click, and select "Open with";
 - Choose Google Chrome to open it.

Isn't it simple! Go give it a try!

**Note**: You need to grant camera access permissions to Google Chrome.
